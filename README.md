# Tech Stack Notes: AI Tools Landscape (Brown Bag 2026)

This repository contains working notes and source artifacts for a graduate-level brown bag talk on the AI coding tools landscape.
It is organized for quick reuse: a polished one-pager, the original transcript source, and the seed prompt used to generate material.

## What is in this repo

- `from-cursor/AI_Tools_Landscape_Brown_Bag_2026.md`: curated one-pager with tool comparison and framing for a Master's audience.
- `from-copilot/full_chat_transcript.md`: full conversation transcript used as source material.
- `prompt.md`: short event/promo prompt used to frame the session.
- `ai-logos/`: visual assets related to tools/branding.
- `from-chatGPT/`: supporting material from ChatGPT-generated exploration.

## How to use these notes

1. Start with `from-cursor/AI_Tools_Landscape_Brown_Bag_2026.md` for the presentation backbone.
2. Pull examples or narrative context from `from-copilot/full_chat_transcript.md`.
3. Reuse or adapt `prompt.md` when promoting a future version of the session.
4. Add your latest findings in a dated section before presenting.

## Update workflow

- Re-validate tool capabilities and pricing shortly before each talk.
- Keep the comparison table focused on workflow differences, not only model quality.
- Add "Last updated" dates in content files when making meaningful revisions.
- Verify any AI-generated claims with current vendor docs and your own tests.

## Notes

The AI coding tool ecosystem changes rapidly. Treat these notes as a living document and prioritize reproducibility, validation, and clear assumptions when sharing recommendations.
