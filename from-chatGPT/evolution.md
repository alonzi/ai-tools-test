🧭 How to Frame This in Your Talk

A useful way to explain the space (this resonates well in brown bags):

1. Three “modes” of AI coding tools
Autocomplete tools → Copilot, Tabnine
AI-native IDEs → Cursor, Replit
Agentic systems → Claude Code, Devin

👉 The key shift: tools are moving from suggesting code → doing tasks

2. Key trend (worth highlighting)
Tools now operate at repo-level context, not just single files
Many can write, test, and deploy entire features
“Agentic coding” (task-based workflows) is the fastest-growing category
3. Simple takeaway slide

You could summarize with:

“Copilot helps you type faster, Cursor helps you think across files, Claude Code helps you delegate entire tasks.”
